## Project Setup

**Requirements:**
- Java 17+
- MySQL 8+
- IntelliJ or your preferred Java IDE
- Git + GitHub
- Maven

**How to Run:**
1. Clone the repo: [repositoryLink]
2. Create a local database called `pmas`
